# Plan 05-04: Evidence Registry UI

**Phase:** 5 — Evidence, Findings, and Gate P3  
**Status:** not started  
**Features:** F8  

## Goal

Build the Evidence Registry page with the evidence table, Add Evidence Sheet panel, file upload component, coverage bar per objective, and CSV export.

## Tasks

### Task 1 — shadcn components for evidence
- Manually create `src/components/ui/tooltip.tsx` from new-york template (wraps @radix-ui/react-tooltip)
- Manually create `src/components/ui/sheet.tsx` from new-york template (wraps @radix-ui/react-dialog with slide-in CSS variants — @radix-ui/react-sheet does not exist as npm package)

### Task 2 — Evidence list and table
- Create `src/hooks/useEvidence.ts`: fetches `GET /api/engagements/:id/evidence`
- Create `src/components/evidence/EvidenceTable.tsx`:
  - Columns: type badge, source, date received, custodian, sensitivity badge, files count, actions
  - Row click navigates to evidence detail
  - Restricted badge shown only to privileged roles
- Create `src/pages/EvidenceListPage.tsx`:
  - EvidenceTable with Add button
  - Coverage bar: X objectives with evidence / Y total (from coverage summary)
  - CSV export button (calls `GET /api/engagements/:id/evidence?format=csv`)

### Task 3 — Add Evidence Sheet panel
- Create `src/components/evidence/AddEvidencePanel.tsx` (shadcn Sheet — slides in from right):
  - Fields: evidence_type (Select), source, date_received (DatePicker), custodian, description, sensitivity (RadioGroup)
  - "Save" button creates evidence item via `POST /api/engagements/:id/evidence`
  - After save: sheet stays open for file upload

### Task 4 — Evidence file upload
- Create `src/components/evidence/EvidenceFileUpload.tsx`:
  - Drag-and-drop + file picker (same pattern as IntakeFileUpload)
  - Uses raw fetch() with FormData (NOT api.ts wrapper)
  - Calls `POST /api/engagements/:id/evidence/:evidenceId/files`
  - Lists existing files from `GET /api/engagements/:id/evidence/:evidenceId/files`

## Acceptance Checks
- [ ] Evidence list shows all non-restricted items to standard users; AL/RO see only standard
- [ ] Add Evidence Sheet opens and creates evidence item
- [ ] File upload works; uploaded filenames appear after save
- [ ] Coverage bar shows accurate objective coverage count
- [ ] CSV export downloads correct file
